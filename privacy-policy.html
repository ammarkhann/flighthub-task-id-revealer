Privacy Policy — FlightHub 2 Task ID Revealer

Last updated: 2026-04-24

This Chrome extension ("the extension") is a browser tool for use with DJI FlightHub 2. This policy describes what the extension does with data.

Summary

The extension does not collect, transmit, store, or share any data about you or your activity. Everything it does happens inside your own browser, in your own session.

What the extension reads

When you are on a DJI FlightHub 2 page and the browser makes a request to the FlightHub API endpoint that returns the task list, the extension reads the response locally. It extracts the task ID, task name, start time, status, and operator name for each task in the response, and displays them in a floating panel inside the same page.

The extension does not:

  • Send any data to any server operated by us or any third party.
  • Store any data in browser storage, cloud storage, cookies, or local files.
  • Use any tracking, analytics, or telemetry service.
  • Access any page other than the DJI FlightHub pages listed in the extension's manifest.

What the extension stores

Nothing. Each time you reload the page, the task list is read fresh from the API response the page itself is making.

Permissions

The extension requests only host permissions for the DJI FlightHub domains declared in its manifest. No other Chrome extension APIs (storage, cookies, history, etc.) are used.

Open source

The full source code of the extension is shipped as part of the extension package. You can inspect it in your Chrome profile's extension folder or by unzipping the installation package.
